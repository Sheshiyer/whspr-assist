## Audio System Detail

### Hearing Enhancement Pipeline

```
Ambient Sound → Dual MEMS Mics → JL7018F DSP →
  ├── ANC (reduce background noise)
  ├── ENC (cancel echo and reverberation)
  └── Frequency Shaping (audiogram-matched)
→ Open-Ear Directional Speakers → Enhanced Audio
```

**Key distinction:** This is not volume amplification. The system identifies and enhances speech frequencies while suppressing non-speech noise. The result is **clearer** sound, not **louder** sound.

### Audiogram Calibration

- Hearing profile uploaded via companion app or Provider Portal
- Frequency response shaped to match individual hearing curve
- Multiple profiles supported (e.g., restaurant, conversation, music)
- Remote recalibration available through Provider Portal

### AI Processing

| Capability | Description |
|------------|-------------|
| **Text Recognition (OCR)** | Reads printed text aloud — labels, menus, signs, documents |
| **Scene Description** | Describes what the camera sees in natural language |
| **Object Identification** | Names objects, products, and items in the field of view |
| **Color Recognition** | Identifies colors for users with reduced color perception |
| **Real-Time Translation** | Coming in future firmware update |

### Privacy

- Cameras activate **only on voice command** ("Hey Cyan, read this")
- No continuous recording
- Visible LED indicator when cameras are active
- Same privacy model as Ray-Ban Meta

## Charging and Battery

| Metric | Value |
|--------|-------|
| Charging Method | Magnetic 2-pin snap connector |
| Charge Time | ~60 minutes (0 to 100%) |
| Battery Life | 4–5 hours active use |
| Standby Time | 72+ hours |
| Battery Type | Lithium polymer |
| Charging Indicator | LED on temple (amber = charging, green = full) |

### Battery Life Context

4–5 hours covers the key social situations where hearing enhancement matters most:

- Family dinner: 1.5–2 hours
- Church service: 1–1.5 hours
- Golf round: 3–4 hours
- Restaurant outing: 1.5–2 hours
- Phone calls: distributed throughout the day

Magnetic snap charging during midday rest extends usable range. Same routine as removing regular glasses.
